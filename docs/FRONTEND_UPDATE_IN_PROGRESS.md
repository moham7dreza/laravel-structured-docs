# Document Creation Frontend Update

Updating the frontend document creation form to match the comprehensive admin panel structure with all tabs:

1. **Basic Information** - Title, description, image, tags
2. **Structure & Category** - Category, structure selection, and dynamic content fields
3. **Branch & Integration** - Git branches and Jira tasks
4. **Permissions** - Editors and reviewers
5. **References & Links** - Document references and external links  
6. **Settings** - Visibility, status, watchers

The form will now match exactly the admin panel's functionality and user experience.
